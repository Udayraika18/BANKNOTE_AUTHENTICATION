# Banknote Authentication

A Streamlit-based machine learning application that predicts whether a banknote
is authentic or fake using statistical features.

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- NumPy

Live link:-
https://banknote-verification-uday.streamlit.app/

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
